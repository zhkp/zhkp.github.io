54. 螺旋矩阵 中等

题目：

给定一个包含 m x n 个元素的矩阵（m 行, n 列），请按照顺时针螺旋顺序，返回矩阵中的所有元素。

题解：

看上去不难，实际上手也不是很难。

不过最好是在纸上做好计算，凭空想很可能想差。

我的做法是右，下，左，上四个方向轮流来，通过方向变量，+ 1 mod 4，switch来实现。

每走完一个方向走过的行数或列数加一，如向左走完，则相当于走过了一行，则走过的行数加一。接着向下走，走完相当于走过了一列，则走过的列数加一。

假设走过的行数为i，走过的列数为j，总行数为m，总列数为n。

  • 向右走列地址范围是[j / 2, n - j / 2)，从小到大，对应行数为i / 2；
  • 向下走行地址范围是[(i + 1) / 2, m - (i - 1) / 2)，从小到大，对应列数为n - j / 2 - 1；
  • 向左走列地址范围是((j - 1) / 2 - 1, n - (j + 1) / 2 - 1]，从大到小，对应行数为m - (i - 1) / 2 - 1；
  • 向上走行地址范围是(i / 2 - 1, m - i / 2 - 1]，从大到小，对应列数为(j - 1) / 2；

注意：判断空集
